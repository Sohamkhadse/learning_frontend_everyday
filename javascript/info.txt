the data type automatically.

Javascript is used in web scripting, server-side development (REST APIs), mobile applications (using React Native), 
and desktop applications.

We can run Javascript in 2 ways:

1.using a client web browser (client-side)
2.using Node.js (server-side)

there are various framework and libraries that are use in javascriptto create mobile apps and servers.

Javascript is an event-driven language, it works on user actions like click, hover, input, etc

Node.js is a runtime environment that allows you to run JavaScript outside the browser.
Normally, JavaScript runs only in browsers, but Node.js lets you run it on the server (backend)

It is built on Google Chrome V8 engine, which makes it fast.
Node.js is important because it makes JavaScript a complete full-stack language,
 allowing developers to build both frontend and backend using the same language.
